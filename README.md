# CRUD APP , React + TypeScript + Vite + Antd

Install Typescript (skip if you already have it )

```bash
npm i -g typescript

```

Install `pnpm` (skip if you already have it )

```bash
npm i -g pnpm
```

Install packages

```bash
pnpm i
```

Run The project

```bash
pnpm run dev
```
